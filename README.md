# ペット相談専門アプリ＜PTCN＞

## アプリ概要
ペット相談専門アプリ＜PTCN＞は、ペットに関する悩みを気軽に相談できるWebアプリケーションです。

ペットの飼い主が質問を投稿し、専門家が回答することで、信頼性の高いアドバイスを受けることができます。

犬や猫だけでなく、その他の動物にも対応しており、画像を添付してより具体的な相談を行うこともできます。

## アプリの使い方

### 一般ユーザー
- 新規登録・ログイン
- ペットに関する質問を投稿
- 画像を添付して相談
- 他のユーザーの質問や回答を閲覧
- 自分の質問履歴を確認

### 専門家
- 管理者が登録した専門家アカウントでログイン
- 投稿された質問に回答
- 自分の回答履歴を確認

### 管理者
- 専門家アカウントの登録

---

## 作成した背景

私は猫を飼っており、日常の中で「ちょっと気になることを気軽に相談できるサービスがあれば便利だな」と感じることがありました。

インターネット上には多くの情報がありますが、情報の信頼性に不安を感じることもあります。

そこで、専門家のみが回答できる仕組みにすることで、安心して利用できるペット相談サービスを作成しました。

---

## 工夫したところ

### 1. 使いやすさを重視したUI設計
常にヘッダーに主要なリンク（質問一覧、質問する、マイページなど）を表示し、どの画面からでもスムーズに移動できるようにしました。

### 2. 検索・並び替えの操作性向上
並び順の変更時に検索ボタンを押さなくても自動で反映されるようにし、より直感的に利用できるよう工夫しました。

### 3. 権限ごとの機能制御
一般ユーザー、専門家、管理者の3つの役割を設け、それぞれに応じた機能のみ利用できるようにしました。

### 4. 画像付き相談への対応
質問に画像を添付できるようにし、症状や状況をより具体的に伝えられるようにしました。

### 5. 入力エラーの日本語化
バリデーションメッセージを日本語化し、ユーザーにとって分かりやすい表示にしました。
